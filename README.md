# Stellaris MP Optimized — Synkarion Crew

Приватная сборка модов для Stellaris 4.3 «Cetus» — субботняя партия 3 игроков RU↔TH.

🌐 **Сайт:** https://mokhnatti.github.io/stellaris-mp-optimized/

## Быстро

1. Подписаться на коллекцию: https://steamcommunity.com/sharedfiles/filedetails/?id=3719769152
2. Скачать [Opimizm.json](Opimizm.json) — импортировать в Paradox Launcher (29 модов в правильном порядке)
3. Сверить checksum в правом нижнем углу лаунчера
4. Подробный гайд для хоста: [HOST_GUIDE.md](HOST_GUIDE.md)

## Состав

- 13 геймплейных модов от Synkarion (UI Overhaul, Tasty Maid, SFT, Starbase Strong и т.д.)
- 10 perf-модов (Ariphaos UP, AIGPO, BPU, StarNet AI, Stability Patch и т.д.)
- 3 русификатора
- 3 наших мода: zz_perf_patch (SFT throttle), zz_tasty_static (frozen portraits), zz_ru_pack (русский + fallback)

Версия Stellaris: **Cetus v4.3.5**, эталон checksum: **3e40** (может измениться при hotfix-ах модов).
