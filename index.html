ПОДПИСАТЬСЯ НА КОЛЛЕКЦИЮ → Steam Workshop · одной кнопкой 29 модов

Шаги для каждого игрока

1

Подписаться на коллекцию

Перейти по ссылке коллекции, нажать «Подписаться на все». Steam скачает 29 модов в фоне (~1.5 ГБ). Подождите завершения загрузки.

2

Импортировать Playset (порядок загрузки)

Скачать Opimizm.json ниже, затем в Paradox Launcher:

  • Игровые наборы → ⋯ (три точки) → Импорт игрового набора
  • Выбрать скачанный Opimizm.json
  • Активировать набор «Opimizm»

Все 29 модов автоматически встанут в правильном порядке.

3

Сверить checksum

⚠️ Важно: checksum нужно смотреть ВНУТРИ ИГРЫ (главное меню Stellaris), а не в Paradox Launcher. В лаунчере неправильный checksum — он показывает другую сумму.

Запусти Stellaris с активным Playset «Opimizm». В главном меню Stellaris внизу/снизу справа будет строка:

Версия: Cetus v4.3.5 (3e40)

Где 3e40 — наш текущий эталон.

Hotfix-ы модов: RegiZero обновляет Ultimate Automation почти каждый день. Когда он выпустит hotfix — checksum изменится у всех. В пятницу вечером сверим новый эталон в Discord.

Если checksum не совпадает:

  • Verify integrity Stellaris (Steam → Library → Stellaris → Properties → Local Files)
  • Подождать пока Steam докачает Workshop subscriptions
  • Сверить состав модов с Opimizm.json (должно быть ровно 29 модов)
4

Подготовка системы (для всех)

  • Discord overlay для Stellaris = OFF
  • OneDrive не должен синкать Documents\Paradox Interactive\
  • Antivirus exclusion на Stellaris install + Documents\Paradox Interactive\Stellaris
  • Windows Game Mode = OFF (для гибридных CPU как 12600KF особенно)
  • NVIDIA Control Panel для Stellaris.exe: Power = Maximum Performance, Anti-Lag/Reflex = OFF
  • Steam Region = Stockholm (для РФ) или Tokyo (для Тая)

Подробный гайд для хоста (i7-9700K) — в файле HOST_GUIDE.md.

5

В субботу — старт

  • За 30 мин до партии — все 5 онлайн в Discord
  • Закрыть лишнее (браузер, OBS, тяжёлые программы)
  • Запустить Steam, дождаться обновлений Workshop
  • Сверить checksum в Discord — у всех 5 одинаковый
  • Хост (9700K) создаёт MP-лобби через Steam Friends
  • Speed 3-4 в late-game (после 2300), не 5
  • Если у тебя CPU слабее 9700K → speed 2 в late-game (иначе тормозишь всех)
Lockstep правило: в Stellaris MP симуляция дублируется на каждом клиенте — самый слабый CPU задаёт скорость для всех. Хостит i7-9700K (предположительно слабейший). Если у тебя слабее — играй на низкой скорости в late-game.
