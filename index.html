Шаг 3 — Сверка checksum

3

Убедиться что у тебя такая же сборка как у других игроков

⚠️ ВАЖНО: checksum нужно смотреть ВНУТРИ ИГРЫ (главное меню Stellaris), а НЕ в Paradox Launcher! В лаунчере показывается checksum чистой игры без модов — это другая сумма и для MP она не подходит.

3.1 В Paradox Launcher с активным Playset «Opimizm» нажми зелёную кнопку «Играть» / «Play» внизу справа.

3.2 Дождись загрузки. Stellaris откроет главное меню (с кнопками «Новая игра», «Загрузить», «Мультиплеер» и т.д.).

3.3 Внизу или сверху в углу главного меню найди строку с версией и checksum:

3.4 Эталон сейчас:

Версия: Cetus v4.3.5 (3e40)

Если у тебя 3e40 — всё ок, ты в одной сборке с остальными.

Hotfix-ы модов: RegiZero обновляет «Ultimate Automation 4.3» почти каждый день. Когда он выпускает hotfix — checksum изменяется у всех. Мы в пятницу вечером сверим новый эталон в Discord. До этого момента эталон может отличаться.
Если checksum НЕ совпадает с эталоном:
  1. Verify integrity Stellaris (Steam → Library → Stellaris → Properties → Local Files → Verify integrity)
  2. Подожди пока Steam докачает все Workshop subscriptions (Шаг 1.4)
  3. Сверь количество модов в наборе — должно быть ровно 29
  4. Сверь набор DLC — должен совпадать с другими игроками (см. Шаг 4)
  5. Импортируй Opimizm.json заново

Шаг 4 — Подготовка системы

4

Системные настройки чтобы не лагало в late-game

Эти настройки — для всех 5 игроков. Времени займут 5 минут.

4.1 Discord

Открой Discord → User Settings (шестерёнка слева внизу) → Game Activity / Игровой оверлей:

  • Найти Stellaris в списке
  • Переключатель «Overlay» = OFF

4.2 Windows Game Mode (Win 10/11)

Settings (Win+I) → Gaming → Game Mode = OFF.

На гибридных CPU (например Intel 12-14 поколения) Game Mode может перекидывать поток на E-cores. Лучше выключить.

4.3 Antivirus exception

Чтобы Defender не сканировал каждое автосохранение Stellaris:

  • Settings → Update & Security / Privacy & Security → Windows Security → Virus & threat protection
  • Manage settings → Add or remove exclusions
  • Add exclusion → Folder:
    • C:\Program Files (x86)\Steam\steamapps\common\Stellaris
    • C:\Users\<ТВОЁ_ИМЯ>\Documents\Paradox Interactive\Stellaris
  • Add exclusion → Process → stellaris.exe

4.4 OneDrive

Если синкается папка Documents с OneDrive — выключи синк для Documents\Paradox Interactive\:

  • Кликни иконку OneDrive в трее → Settings → Backup → Manage Backup
  • Если Documents включён — выключи. Иначе автосохранения Stellaris будут двойной нагрузкой

4.5 Power Plan

Win+R → powercfg.cpl → High Performance (Высокая производительность).

4.6 NVIDIA Control Panel (для GeForce)

  • Правый клик на рабочем столе → NVIDIA Control Panel
  • Manage 3D settings → Program Settings → Add → выбери stellaris.exe
  • Power management mode = Prefer maximum performance
  • Low Latency Mode = Off (это Reflex/Anti-Lag, для grand strategy не нужно)
  • Vertical sync = Off

4.7 settings.txt

После первого запуска игры закрой её, открой блокнотом:

C:\Users\<ТВОЁ_ИМЯ>\Documents\Paradox Interactive\Stellaris\settings.txt

Найди и поменяй:

vsync=no
multi_sampling=0
shadowSize=0
maxanisotropy=0
autosave=4

autosave=4 = раз в год вместо раз в месяц = меньше фризов в late-game.

4.8 Steam Cloud Save

Steam Library → Stellaris → правый клик → Properties → General → переключатель «Keep games saves in the Steam Cloud» = OFF.

4.9 (Опционально для хоста) Process Lasso

Только если у тебя Intel 12-14 поколения с E-cores (например i5-12600KF). Иначе можно пропустить.

  • Скачай: bitsum.com/processlasso
  • Установи, найди в списке stellaris.exe
  • CPU Affinity → Always → оставить только cores 0-11 (P-cores), снять 12-15 (E-cores)

Шаг 5 — В субботу старт

5

Что делать в день партии

За 30 минут до партии

  1. Все 5 онлайн в Discord
  2. Закрыть лишнее: браузер с тяжёлыми вкладками, OBS, тяжёлые программы (видеоредактор, IDE)
  3. Запустить Steam → подождать пока докачаются обновления Workshop
  4. Запустить Stellaris через Paradox Launcher (Playset «Opimizm»)
  5. В главном меню сверить checksum — у всех 5 должен быть одинаковый
  6. Если у кого-то другой → обновить Workshop / Verify integrity

Создание лобби (хост)

  1. Главное меню → Multiplayer
  2. Host Game
  3. Galaxy settings — обсудить компанией перед стартом
  4. Создать лобби — пригласить друзей через Steam Friends

Подключение клиентов

  1. Friends list в Steam — увидите хоста с пометкой «играет в Stellaris»
  2. Правый клик по нику хоста → Join Game
  3. Или: Multiplayer → Browse → войти по приглашению

Во время игры

  • Speed 3-4 в late-game (после 2300 года), не 5
  • Если у тебя CPU слабее i7-9700K → speed 2 в late-game (иначе тормозишь всех)
  • Не открывай тяжёлые UI окна во время монтлый-tick
  • Autosave раз в год — это норма, 1-2 секунды фриз
  • При спайке лагов — пауза 30 сек, продолжайте
Lockstep правило: в Stellaris MP симуляция дублируется на каждом клиенте — самый слабый CPU задаёт скорость для всех. Хостит i7-9700K (предположительно слабейший). Если у тебя слабее — играй на низкой скорости в late-game.

Что делать если что-то не работает

Не вижу мода в Playset после импорта

Steam ещё не докачал. Открой Steam → Downloads — посмотри прогресс. Для проверки: Steam → Library → Stellaris → Properties → Workshop → должно быть ~1.5 GB. Если меньше — Verify integrity.

Checksum не совпадает с эталоном
  • Сделай Verify integrity
  • Подожди пока Steam докачает все Workshop subscriptions
  • Сверь набор DLC — у всех должен быть одинаковый
  • Импортируй Opimizm.json заново
  • Если RegiZero выпустил hotfix Ultimate Automation — у всех изменится. Сверим новый эталон в Discord
OOS (Out Of Sync) посреди игры
  1. Все игроки → Main Menu (НЕ desktop)
  2. Хост грузит autosave (НЕ manual save)
  3. Все возвращаются
  4. Если повторяется — удали папку Documents\Paradox Interactive\Stellaris\oos\ у всех
  5. Откат на autosave −1 год
CTD (Crash To Desktop)
  • Лог: Documents\Paradox Interactive\Stellaris\logs\error.log — кинь в Discord, посмотрим
  • В late-game (после 2400) бывает у всех — возможно crisis spike
  • Грузим autosave −1 год от падения
Сильно лагает на speed 5
  • В late-game это норма — все играют на speed 3-4
  • Если у тебя слабый CPU — speed 2 в late-game
  • Если жопа — хост через консоль чистит галактику: kill_country [id] мёртвых AI империй

Стресс-тест (на Ryzen 5800X3D)

Эталонный замер на Huge 1000 stars / 16 AI / 4 FE / 3 Marauder + Khan / Crisis 5× / year 2469 (full late-late-game):

  • Daily tick avg: 117 ms (зелёная зона <150)
  • one_year: 92.4 секунды
  • FPS: ~250 (4ms frame)
  • Errors: 150 строк за всю сессию (vanilla DLC bugs, наши моды чисты)

На i7-9700K хосте ожидается +30-50% к этим числам — всё ещё играбельно до 2400.
